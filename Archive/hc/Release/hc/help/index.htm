“HC.D2H”是一个D2Hackit外挂模块，需要D2Hackit 110（或更高版本）才能运行。该模块的主要功能是在角色遇到各种威胁时自动做出快速反应，同时也提供了如快速退回聊天频道、快速用Town Portal回城、快速客户端重绘（消除“黑墙”状态）等等便利有效的辅助功能。

使用方法：

  1. 下载“hc.zip”并解压，将所有的文件复制到D2Hackit 110所在的目录里并覆盖源文件。
  2. 完成了上面的步骤后，请确认你的D2Hackit 110目录里至少有这些文件（一个都不能缺）：

     
  3. 运行Diablo II。
  4. 用Alt-Tab将游戏最小化，双击“loader.exe”启动D2Hackit 110。
  5. 角色进入游戏后，确认您在游戏屏幕上看到了“ Module 'hc' by Abin loaded!  ”这行提示。
  6. 参考程序设置与相关命令进行设置。程序会将设置存盘，因此每个角色只需要设置一次。
  7. 注意：某些重要的功能（例如自动喝药水等等）要求用户在角色进入游戏以前就已经加载本模块，否则只有当角色进入一个新游戏以后这些功能才能被正常使用．

功能简介：

  1. 逃避功能：
     
    当角色生命值低于特定值或遭到Hostile时自动迅速采取逃避措施，包括退回聊天频道、用TP回城，和用TP回城并对敌对玩家进行谩骂。
  2. 自动喝药水功能：

    当角色的生命值或魔法值低于设定的百分比时，自动喝药水，当雇佣兵生命值低于设定的百分比时，自动给它喂药水。
      
  3. 同步功能：
     
    迫使客户端的显示与服务器端同步。简单地说，当角色遇到“黑墙”，砍怪物没反应，不知道自己实际上到底在哪里时（也许在几个屏幕之外），只要按某一个设定的键，客户端立即以角色真实坐标进行重绘，于是角色即回到实际所在的真实位置，也就是服务器端角色位置。注意：角色通过TP、WP等方式切换场景后，请稍候数秒才能使用同步功能，否则在lag严重的时候可能引起游戏掉线。
     
  4. 快捷键功能：
     
    一键退出：按一个键，角色立即退回聊天频道；
    一键回城：按一个键，角色立即开一个Town portal并自动进入；
    一键同步：按一个键，立即实现客户端重绘。
     
    允许对“退回聊天频道”、“用TP回城”和“同步功能”设定任意快捷键。以后要实现这些功能只需按一个健即可。例如，您可以设置“退回聊天频道”的快捷键为“F9”，“用TP回城”的快捷键为“F10”，“同步功能”的快捷键为“F11”。需要退出游戏时只需按一下“F9”，角色立即退出游戏回到聊天频道；而如果遇到了黑墙或丢包现象，只需按一下“F10”就全搞定；在一大堆怪物的包围中要想顺利地打开Town Portal并用鼠标点中它不容易吧？现在好了，按一下“F11”你就回到城里，再也不用手忙脚乱了。酷吧？

角色低生命值反应设置

功能 角色低生命值反应设置
语法   .hc  hp  <quit / town / potion>  <life-percent> 
快捷键 无
参数说明 Quit:退出游戏；Town:TP回城；Potion:喝紫/红药水
<life-percent>：生命警戒值百分比 (0到100)

注:你可以多重设置,例如可以设定当生命值低于80%喝药水,低于60%回城,低于25%退出游戏,三项设定可以共同作用,互不冲突.如下例:
示例   .hc  hp  potion  80   

  .hc  hp  town  60   

  .hc  hp  quit  25   

当生命值低于80%喝药水,如果喝药水失败或者喝了药水以后生命值继续下降,降至低于60%时回城,降至低于25%时退出游戏
